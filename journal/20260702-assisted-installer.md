# OpenShift fundamentals

## Install OpenShift with Assisted Installer
## Prerequisites
* set DNS with api.[clustername].domain.local and *.apps.[clustername].domain.local

## Console
* Go to console.redhat.com, select Red Hat OpenShift
* Select Create cluster on Red Hat OpenShift Container Platform
* Go to datacenter and click create cluster
* Fill clustername and base domain
* Add host to create ISO
* Run installation on VM and see progress in console
* Take note of the kubeadmin password
